# Phigros Data Farmer

本项目是一款专为Phigros设计的自动循环刷Data工具。

本项目基于 [Xizo-114514/Phigros-Script](https://github.com/Xizo-114514/Phigros-Script/) 进行二次开发与深度优化。

### 🌟 核心改进 (Revision by lemocn)

相比于原版脚本，本项目进行了以下核心修复与功能增强：

* **完美循环对齐**：修复了结算后自动开始下一轮时，因加载状态不一致导致的严重 **Offset 偏移**问题。
* **自动化对齐逻辑**：循环中自动执行“双击暂停+重新开始”，强制同步每一局的延迟，确保长效挂机不偏移。

### 🚀 快速开始
1. **安装Python环境**：
运行`1.建议使用该python版本（记得选择配置PATH）python-3.11.9-amd64.exe`

2. **安装依赖**：
运行 `2.点我安装依赖.bat` 或执行：
```bash
pip install -r requirements.txt

```

3. **启动脚本**：
运行 `3.点我运行.py`，根据界面提示连接设备并开始挂机。

###Tips:
强烈建议使用模拟器并用较低分辨率（推荐MuMu模拟器,分辨率 960 x 536）进行刷取，稳定且节约性能开销。
强烈建议在游戏中启用低分辨率模式，进一步提升稳定性及节约性能开销。
